
# Number Guessing Game

## Overview
This is a simple Python terminal game where the player tries to guess a randomly generated number. The game demonstrates basic Python concepts including loops, conditionals, user input, and simple logic.

## Features
- Choose a difficulty level (easy, medium, hard)
- Limited attempts based on difficulty
- Hints provided halfway through attempts
- Replay option after winning or losing

## How to Play
1. Run the `game.py` file in a Python terminal.
2. Follow the prompts to guess the number.
3. Try to guess the number in as few attempts as possible.
